## Insert Date String

Dengan memencet `Ctrl` `Shift` `i`, maka akan tampil waktu saat ini. Contoh: `2023-01-30 12:05:12`.

## Relative Path

Dengan memencet `Ctrl` `Shift` `h`, maka akan tampil list semua file pada project yang terbuka. Ketika kita memencet salah satu isi list, maka akan tercetak relative pathnya. Contoh: `../../../public/gambar/error.png`.
